## 🎮 The Game Flow

**Page 1 — Welcome**: Shows your "Bhavana & Mahender" cover with animated Start button below.

**Page 2 — 4×4 Memory Game**: 
- 16 cards in a 4×4 grid (8 pairs)
- Uses ALL 8 of your images
- Mobile-optimized — works perfectly on phones in portrait mode
- Live moves counter & timer
- Mute button for the music
- Bhavana & Mahender signature at the bottom

**Page 3 — Save the Date Reveal**: 
- Confetti animation when all pairs matched
- Wedding details displayed elegantly
- WhatsApp share button

## 🚀 Deploy to GitHub Pages

1. Create new repo (Public) on github.com
2. Upload ALL these files:
   - `index.html`, `style.css`, `script.js`
   - `template1.png`, `template2.png`, `template3.png`
   - `audio1.mp3`
   - `image1.jpg` to `image8.jpg` (all 8!)
3. Settings → Pages → main branch → Save
4. Wait 2 min → Get your link: `https://YOUR-USERNAME.github.io/REPO-NAME/`
5. Share via WhatsApp! 

## 📱 Mobile-Friendly Features

- ✅ Touch-optimized cards (proper size for fingers)
- ✅ Responsive 4×4 grid adapts to any screen
- ✅ No zooming/pinching needed
- ✅ Works in portrait AND landscape
- ✅ Audio works on mobile browsers
- ✅ Fast loading with lazy-loaded images

## 🖼️ Image Specs

- **Format**: JPG (smaller file size)
- **Size**: 500×500px (square works best for 4×4 grid)
- **File size**: Under 200KB each (compress with [TinyPNG](https://tinypng.com))
- **Suggestions**:
  - image1: Bride photo
  - image2: Groom photo
  - image3: Ring 💍
  - image4: Broom 🧹
  - image5: Logo / monogram
  - image6: Engagement photo
  - image7: Couple photo
  - image8: Venue / favorite memory

## 🎵 Audio Specs

- Filename: `audio1.mp3`
- Duration: 2-3 mins (loops automatically)
- File size: Under 3MB

## ✏️ Customize Details

Edit `index.html` to change wedding info in the invite section.

## 🔧 Troubleshooting

- **Music doesn't auto-play?** Browser policy — user must click Start (which is a user interaction)
- **Cards too small/big on mobile?** Already optimized for various screen sizes
- **Want fewer cards?** Edit `script.js` line 8-17 to remove some image entries
